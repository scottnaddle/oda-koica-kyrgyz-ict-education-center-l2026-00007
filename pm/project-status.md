# PM Project Status

프로젝트: Kyrgyz ICT Education Center Design & Author Supervision
상태: Initiation / pre-kickoff alignment

## Summary
- 본 프로젝트는 수주 가정 기준으로 수행계획서와 WBS 초안이 생성된 상태다.
- 현재 핵심은 기술 수행보다 **범위/역할/승인체계/파트너 구조 정렬**이다.
- 초기 리스크는 license 구조, bilingual deliverables, 파트너 역할 경계, 발주처 승인 리드타임에 집중되어 있다.

## Schedule
- 현재 단계: 착수 전 정렬
- 다음 핵심 마일스톤:
  - 계약범위 정합성 확인
  - 수행계획서 v0.1 확정
  - 파트너 역할표 확정
  - Design Implementation Plan 초안 제출

## Budget
- 아직 계약 확정본 및 실행 예산이 반영되지 않아 상세 budget baseline은 미확정 상태다.
- 다만 아래 비용 항목은 초기 추적이 필요하다:
  - 내부 PM/QA 인건비
  - 설계/기술 파트너 비용
  - 번역/검수 비용
  - 현지 행정/협의 대응 비용

## Risks / Issues
- 가장 큰 리스크는 Kyrgyz local design license 구조다.
- JV 불가 조건 때문에 일반적인 consortium 가정이 성립하지 않을 수 있다.
- bilingual deliverables는 품질/일정 모두에 영향을 줄 수 있다.
- 계약번호/계약 확정본 미반영 상태는 초기 issue로 관리한다.

## Next Actions
- 계약 범위 및 계약번호 반영
- 발주처 승인체계 확인
- 파트너 역할표 초안 작성
- Design Implementation Plan 준비 시작
- communication log와 budget tracker 초기 세팅
