# Scope / Deliverables Summary

## One-line summary
이 사업은 ICT 교육센터 신축을 위한 **설계 + 입찰지원 + author’s supervision** 패키지다.

## Work scope breakdown
| Track | 내용 | 시사점 |
|---|---|---|
| Preliminary / survey | site investigation, survey, layout/masterplan options | 단순 문서 작업이 아니라 현장/설계 준비 포함 |
| Design development | SD / DD / CD 단계 설계 | full-cycle design capability 필요 |
| Engineering documents | drawings, specs, BOQ, cost estimate, calculations | 설계/엔지니어링 중심 |
| Permits / admin | 관련 인허가 및 행정 준비 | 현지 대응 역량 중요 |
| Bidding support | tender docs, addenda, contractor selection support | 발주 준비 consultant 역할 |
| Author’s supervision | 주간/월간 보고, design clarification, design change 대응 | 시공단계까지 책임 연장 |

## Expected deliverables
- survey / investigation results
- layout options / masterplan
- drawings
- technical specifications
- bill of quantities (BoQ)
- structural calculations
- mechanical / electrical load calculations
- reports
- final deliverables approved by local permitting authorities and KOICA/client
- weekly site inspection report
- monthly progress report
- completion report

## Delivery / contract structure
- Design: 9 months from effective date
- Bidding support + author’s supervision: 15 months (subject to change)
- Contract price remains unchanged even if bidding support / supervision period is extended

## Language / communication
- Deliverables: English and Russian
- Ruling language: English
- Communication language: English

- proposal narrative도 교육 일반론보다 **design consultant capability + project coordination + KOICA/ODA context understanding** 쪽으로 잡아야 한다.
