# Deliverables Structure

이 폴더는 프로젝트 산출물을 단계별로 정리하기 위한 기준 구조다.

## Folder layout
- `01-initiation/` : 착수 및 초기 계획 산출물
- `02-design/` : 설계 단계 산출물
- `03-tender-support/` : 입찰지원 단계 산출물
- `04-supervision/` : 시공지원/author's supervision 산출물
- `99-closeout/` : 종료 및 인계 산출물

## 운영 원칙
- 실제 파일이 생성되면 해당 단계 폴더에 누적한다.
- `deliverables/deliverables-inventory.md`와 실제 파일 경로를 함께 관리한다.
- 제출본과 작업본을 구분해야 하면 하위에 `draft/`, `review/`, `final/` 구조를 추가할 수 있다.
